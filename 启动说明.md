# 项目启动说明

> 文档版本：v1.1.0  
> 最后更新：2026-02-10

## 1. 环境要求
- Android Studio（最新稳定版）
- JDK 17（可直接使用 Android Studio 自带 JBR）
- Android SDK（项目默认路径或本机已配置路径）
- 真机或模拟器（Android 10+）

## 2. 打开项目
1. 打开 Android Studio
2. 选择 `Open`
3. 打开目录 `D:\Video Downloader`
4. 等待 Gradle Sync 完成

## 3. 终端构建（推荐）
```powershell
.\环境一键配置.ps1
.\gradlew.bat :app:assembleDebug
```

产物位置：`app/build/outputs/apk/debug/app-debug.apk`

## 4. 安装到设备
```powershell
.\gradlew.bat :app:installDebug
```

## 5. 最小验证流程
1. 首页粘贴抖音分享文案并解析。
2. 选择可下载项并开始下载，确认队列状态更新。
3. 测试 X 链接前先在“X 设置”保存有效 Cookie。
4. 对 m3u8 多选项：
   - 看是否出现推荐项（仅在有质量信息时）。
   - 无质量信息时是否出现“预览”按钮。
5. 下载完成后在“历史/已完成”确认可播放。

## 6. 常见提示
- 如果 Android Studio 一直下载 `gradle-*-src.zip`：这是源码包下载，不影响命令行构建与安装。
- 如果 X 解析失败：优先检查 Cookie 是否失效、代理是否可用。

## 7. 一键清理

